# 📺 YouTube Clone Backend (MERN)

This is the **backend server** for a YouTube-like video streaming platform.  
Built with **Node.js**, **Express**, and **MongoDB**, it supports full CRUD operations for videos, user authentication, subscriptions, and secure media handling.

## ✨ Features

- 🔐 **User Authentication** with JWT (access & refresh tokens)
- 📤 **Video Uploading** via Multer
- 📺 **Video Streaming & Listing**
- 🧾 **Subscriptions** between users
- ❤️ **Likes & Dislikes**
- 💬 **Comment System**
- 🔁 **Token Refresh & Protected Routes**
- 🛠️ Custom API Response + Error Handling
- 📦 Ready for production (dotenv, logging, structured folders)

---

## 🛠 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB** + **Mongoose**
- **JWT** (Access & Refresh tokens)
- **Multer** (File upload)
- **Bcrypt.js** (Password hashing)
- **Postman** for API testing
